Actividad 1: Desarrollo Web

Tema 1: Elementos de Páginas Web

Orígenes de la Web

La Web (World Wide Web) fue creada por Tim Berners-Lee en 1989. Su objetivo era facilitar el intercambio de información entre científicos mediante una plataforma global accesible desde cualquier parte del mundo. La primera página web fue lanzada en 1991, basada en HTML.

Web 2.0

Se refiere a la evolución de la Web hacia plataformas interactivas donde los usuarios pueden generar contenido, como redes sociales y blogs. En lugar de ser solo lectores, los usuarios pueden contribuir activamente.

Web 3.0

Es la futura evolución de la web que se basa en tecnologías como la inteligencia artificial, blockchain y el Internet de las cosas (IoT). Se caracteriza por ser más semántica, permitiendo una comprensión más profunda de los datos por parte de las máquinas.

La W3C y los estándares

El World Wide Web Consortium (W3C) es una organización internacional que establece estándares para asegurar que el desarrollo de la web sea accesible, interoperable y funcione en múltiples plataformas. Algunos estándares incluyen HTML, CSS, XML y otros.

Lenguajes de etiquetas

Son lenguajes usados para definir el contenido de una página web. Los lenguajes de etiquetas más comunes son HTML, XML y XHTML.

SGML

El Standard Generalized Markup Language (SGML) es un estándar para la definición de lenguajes de marcado como HTML y XML.

XML

Extensible Markup Language (XML) es un lenguaje de etiquetas que permite la definición de estructuras de datos, útil para almacenar y transportar información.

HTML

El Hypertext Markup Language (HTML) es el estándar para estructurar contenido en la web mediante el uso de etiquetas.

XHTML

Extensible Hypertext Markup Language (XHTML) es una versión de HTML que sigue las normas de XML, más estricta en cuanto a sintaxis.

HTML5

Es la versión más reciente de HTML. Introdujo nuevas etiquetas semánticas, APIs para gráficos y multimedia, y mejor soporte para aplicaciones web.

Otros lenguajes relacionados al desarrollo web

Además de HTML, se utilizan otros lenguajes como PHP, Ruby, Python y SQL para el desarrollo del lado del servidor.

CSS/CSS3

CSS es el lenguaje de hojas de estilo que define cómo se ve un sitio web. CSS3 es la evolución de CSS, con nuevas propiedades para animaciones, efectos y diseño responsivo.

JavaScript

Es un lenguaje de programación que permite interactividad en la web, como formularios dinámicos, animaciones, y manejo del DOM.

Ajax

Asynchronous JavaScript and XML (Ajax) es una técnica para crear aplicaciones web interactivas y dinámicas que pueden cargar datos en segundo plano sin recargar la página completa.

Tecnologías y lenguajes del lado del servidor

Incluyen lenguajes como PHP, Node.js, Python y bases de datos como MySQL, que permiten gestionar el contenido del lado del servidor.

Navegadores

Son programas que permiten a los usuarios acceder a la web. Algunos ejemplos son Chrome, Firefox, Safari y Edge.

Motores renderizados

Son los encargados de interpretar el código de las páginas web para mostrar su contenido correctamente. Ejemplos incluyen WebKit, Blink, y Gecko.

Herramientas de desarrollo

Son utilidades que ayudan a los desarrolladores a crear, probar y depurar sitios web. Pueden ser extensiones de navegadores como Chrome DevTools o software como Visual Studio Code.

Software WYSIWYG

WYSIWYG significa "What You See Is What You Get". Son editores visuales que permiten crear páginas web sin escribir código, como Dreamweaver y WordPress.

Otras alternativas

  • Figma: Herramienta para diseño colaborativo de interfaces.
  • InVision: Plataforma para la creación de prototipos interactivos web.
  • Zeplin: Herramienta que conecta el diseño con los desarrolladores.
  • GitHub: Plataforma para control de versiones y colaboración en proyectos de código.
  • Inspección de código desde el navegador: Consiste en usar herramientas del navegador para revisar, modificar y depurar el código de una página web en tiempo real, como la inspección de elementos en Chrome.

Tema 2: El Diseño De Una Página Web

Tipos de páginas Web

  • Páginas web estáticas: No permiten interacción con el usuario, su contenido es fijo.
  • Páginas web dinámicas: Son interactivas, y su contenido puede cambiar según la interacción del usuario o la base de datos.
  • Landing pages: Páginas diseñadas específicamente para convertir visitantes en leads o clientes.

Puntos básicos en la elaboración de una página Web

Incluyen el diseño responsivo, navegación clara, contenido accesible, tiempos de carga rápidos y optimización para SEO.

Tema 3: Estructura De Una Página Web

Jerarquía de directorio de un proyecto web

Un proyecto web tiene carpetas organizadas por funciones. Por ejemplo:

  • root/: Directorio principal.
  • css/: Archivos de estilo.
  • js/: Scripts JavaScript.
  • images/: Imágenes usadas en el sitio.

Elementos de estructura para el diseño de una página Web: Concepto

Incluye el uso de etiquetas semánticas en HTML5, como <header>, <nav>, <main>, <footer>, para organizar el contenido de manera lógica y accesible.

Regla utilizada para validar y certificar una página web

La W3C ofrece un validador de HTML y CSS que asegura que el código sigue los estándares web. Esto garantiza la accesibilidad y el correcto funcionamiento de la página en diferentes navegadores.

Reglas para escribir HTML

  • Usar etiquetas correctamente anidadas.
  • Utilizar atributos en minúsculas.
  • Cerrar todas las etiquetas.

Estructura básica de un archivo web

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la página</title>
</head>
<body>
    <header>Encabezado</header>
    <main>Contenido principal</main>
    <footer>Pie de página</footer>
</body>
</html>
